# Multi-Agent Core Utilities
This package houses shared Python utilities required by the core engine and specialized feature packages.

## Intended Components
- **LLM Connectors**: Standardized wrappers for VertexAI, Mistral, and AWS Bedrock.
- **Workflow Parsers**: Logic to validate and ingest the JSON instruction sets.
- **IO Utilities**: Refinements to the `FileWriter` and `FileReader` for agentic context management.
