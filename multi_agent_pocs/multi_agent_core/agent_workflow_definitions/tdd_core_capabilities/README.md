# TDD Core Capabilities
This directory contains the JSON-driven instructional blocks for the AI-agentic TDD cycle.

## Components
- **red_green_refactor_logics/**: Probabilistic and deterministic rules for the three stages of TDD.
- **regression_check_standards/**: Final validation logic before code is staged for commit.
- **security_guardrails/**: Authority and permissioning standards for agent labor.

## Usage
These JSON blocks are "called" by the orchestrator to provide agents with the context required to perform high-fidelity coding tasks.
