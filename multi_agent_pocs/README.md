# Multi-Agent Orchestration Service
This directory serves as the namespace for a modular, enterprise-grade AI-agentic SDLC service.

## Architecture
- **multi_agent_core/**: The foundational engine, shared registries, and instructional TDD framework.
- **modularize_3_coding_agents_specialized_roles/**: A specific feature implementation of the core engine focusing on specialized agent roles.

## Design Philosophy
We use a **Mechanical Engine** (Python Orchestrators) driven by an **Instructional Engine** (JSON Capability Blocks) to automate software development lifecycles.
